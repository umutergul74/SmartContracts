# Arbitrum target

This directory contains reviewed seed metadata for the Arbitrum Immunefi program. The seed is
never treated as permanently current. Run `scbounty scope check arbitrum` immediately before
every real-target analysis.

The first analysis profile is intentionally limited to bridge, gateway, cross-domain messaging,
and accounting code. Compiling a repository for dependency resolution does not place every file
in bounty scope; findings are admitted only when their source path maps to the live scope.
